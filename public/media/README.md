# Media Assets Structure

Organize your portfolio content here:

```
media/
├── hero/
│   ├── desktop.mp4     # 16:9 horizontal (1920x1080)
│   └── mobile.mp4      # 9:16 vertical (1080x1920)
│
├── photo/
│   ├── dance/
│   ├── wedding/
│   ├── kids/
│   ├── brand/
│   └── custom/
│
└── video/
    ├── dance/
    ├── wedding/
    ├── kids/
    ├── brand/
    └── custom/
```

## Naming Convention

- Use descriptive names: `wedding-anna-dmitro-01.jpg`
- Thumbnails for videos: same name as video but `.jpg`
